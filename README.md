## i3

[i3](https://i3wm.org) is a tiling window manager and I use it as my daily WM.

### Installation

`pacman -S i3-wm`

### Configuration

`./config/i3/install`

## Qute Browser

[Qute Browser](https://qutebrowser.org) is a Vim-style Web Browser.

### Installation

`pacman -S qutebrowser`

### Configuration

`./config/qutebrowser/install`
